<h3 id="objetivo">🎯 Goals</h3>

- Getting better at programming in the only way it works...practicing
- In This project I forced myself to solve this problem without using array, but the most difcult thing was to choose a way to make a alert(not using the method alert) right above the <code>&lt;label&gt;</code> without creating a strange space between the <code>&lt;label&gt;</code> and the <code>&lt;input&gt;</code>, and make it disapear when the user for exemple reset. First thougth was to make a function, but in the end my solution was more simple, I added a <code>&lt;span&gt;</code>  to the <code>&lt;label&gt;</code> and by javascript added a text. I learn a lot.
